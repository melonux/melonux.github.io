摘要

罗列一些重要的统计学概念，简述概念的含义与用法，作为日后参考使用。

1. 数据 (Data)

数据是一个通用术语，指我们观察到的信息。数据有不同的类型：

  • 定量数据 (Quantitative data)：数值型的数据。有很多种。

    • 离散数据 (Discrete data)：比如用整数表示的数据。

      • 二元数据 (Binary data)：一类重要的离散数据，因为数值 0,1 可以表示 \text{True}, \text{False} 或者 \text{On}, \text{Off}。

      • 类别数据 (Categorical data)：可以看作是二元数据的进一步扩展，比如用 1,2,3 分别表示 A,B,C 三个不同的组或类别。

    • 连续数据 (Continuous data)：常用连续的实数表示的数据，比如身高、体重、等待某事件发生所需的时间 (Time-to-event data) 等。

  • 定性数据 (Qualitative data)：文字型的数据。

2. 随机性 (Randomness)

随机性源于我们对世界的无知。即使世界是由一个固定的、确定性的伪随机算法生成的，只要我们无法获取算法的种子（seed）或完整的规则，世界在我们看来就是随机的。也就是说，随机性并非一定是宇宙本身固有的，而是我们对宇宙理解的限制所导致的。

几乎所有数据都受到随机性的影响，阻止我们做出完美预测，也让我们无法在赌场100%赢钱。这引出了一个重要问题：在随机性存在的情况下，我们如何从数据中学习？

3. 统计学 (Statistics)

统计学可以被视为研究数据的科学，包括数据的收集、分析与展示。统计学家的目标是透过数据中的随机性，发现其背后的结构，并将其转化为有用的信息。掌握统计学的人可以通过分析、处理数据而获得丰厚的回报。1

4. 概率论 (Probability theory)

随机性往往跟混沌（Chaotic）、不受控制（Uncontrolled）联系在一起，概率论则是研究随机性的内在结构的科学，让人们可以理解并运用随机性。它与统计学的区别在于，概率论是一个数学分支，是基于一些前提假设建构起来的逻辑大厦，其理论本身的发展并不需要实验数据。而统计学是一门实证性的科学，必须先收集到数据才能开始研究。概率论是统计学的理论基础。

5. 随机变量 (Random variable)

概率论中，用于表示随机数据的核心对象就是随机变量。它可以取不同的值，每个值都有不同的概率。随机变量常用 X 表示，而某个具体的取值则用 x 表示。其本质是一个函数，将试验的所有可能结果 (Outcomes) 映射到数值。根据数值类型的不同，随机变量又可分为连续的、离散的、混合的。

6. 概率分布 (Probability distribution)

描述随机变量可能取值的概率，揭示了数据的随机性中隐藏的结构。所有随机变量都具有累计分布函数 (Cumulative distribution functionm, CDF) 。随机变量的类型，由累计分布函数可以导出：

  • 概率密度函数 (Probability density function, PDF)：用于描述连续型随机变量取值的概率。
  • 概率质量函数 (Probability mass function, PMF)：用于描述离散型随机变量取值的概率。

概率分布可以是任何形状的，但有一些常见的分布：

  • 均匀分布 (Uniform distribution)：表示所有取值的可能性都是相等的。
  • 伯努利分布 (Bernoulli distribtion)：用于描述二元数据的概率，比如在游戏中获胜的概率。
  • 二项分布 (Binomial distribution)：描述经过 n 轮游戏后，获胜次数分别为 0,1,\dots,n 的概率。
  • 泊松分布 (Poisson distribution)：描述单位时间内随机事件发生的次数的概率分布。
  • 正态分布 (Normal distribution)：连续的钟形曲线。最常见也是应用最广的分布。

7. 描述性统计量：单一数值描述数据整体的情况。

  • 中心趋势：数据的“中心”在哪里。

    • 均值 (Mean)、期望值 (Expectation)：\mu=E[X]=\int xp(x)dx 。
    • 中位数 (Median)：数据排序后的中间值；CDF等于0.5的位置: F(m)=0.5 。
    • 众数 (Mode)：数据中出现次数最多的项；PDF的最高点；随机变量最可能的取值。

  • 离散程度：数据分散的范围。

    • 方差 (Variance)：\text{Var}[X]=E[(X-\mu)^2]，数据远离均值的程度。
    • 标准差 (Standard deviation)：\sigma=\sqrt{\text{Var}[X]}，以数据原本的单位描述远离均值的程度。

  • 分布形状：

    • 偏度 (Skewness)：描述分布函数偏向某一侧的程度。\gamma_1 = E[(\frac {X-\mu} {\sigma})^3] 。
    • 峰度 (Kurtosis)：描述分布函数的尖锐程度。\text{Kurt}[X]=E[(\frac {X-\mu} {\sigma})^4] 。

8. 随机变量的函数 (Transformation of r.v.)

比如服从标准正态分布的随机变量其平方服从卡方分布：X \sim N(0,1) \implies g(X)=X^2 \sim \chi^2 。

9. 条件概率 (Conditional probability)

P(A|B) 描述随机变量B是如何影响另一个随机变量A的概率的。当两个随机变量相互独立 (Independent)时，有 P(A|B)=P(A)，P(B|A)=P(B) 。

10. 贝叶斯定理 (Bayes’ theorem)

P(\text{Event}|\text{Data}) = \frac {P(\text{Data}|\text{Event})P(\text{Event})} {P(\text{Data})}

描述当观察到试验数据 \text{Data} 后，我们对事件发生概率的先验信念 (Prior belief) 应该如何更新。

11. 贝叶斯主义 (Bayesians) vs 频率主义 (Frequentists)

它们是统计学中两种主要的推断方法，在解释概率、处理数据和解决问题的方式上有本质区别。

比较维度 贝叶斯主义 频率主义
概率的定义 概率是主观信念或对不确定性的度量，反映事件发生的可能性。 概率是长期频率，即事件在无穷多次重复实验中发生的比例。
参数的本质 参数是随机的，具有不确定性，通过概率分布（如先验和后验）进行描述。 参数是固定的（但未知），没有随机性。所有不确定性来源于数据。
数据的作用 数据用于更新先验分布，结合先验信息和数据得到后验分布。 数据是唯一的信息来源，不依赖先验信息，参数估计完全基于样本数据。
推断结果 推断结果是参数的后验分布，提供参数可能值的全面描述。 推断结果是点估计（如最大似然估计）或区间估计（如置信区间）。
假设检验 通过后验概率评估假设是否可信，或使用贝叶斯因子比较假设的相对支持度。 使用 p 值判断零假设是否成立，基于假设成立下数据的极端性来检验假设。
随机性的来源 参数和样本数据都具有随机性，随机性体现在参数分布和数据分布中。 只有样本数据具有随机性，参数是固定的。
计算复杂度 计算复杂，特别是在高维参数空间中，通常需要数值方法（如 MCMC）。 计算较简单，许多方法有解析解（如最大似然估计）。
先验信息的作用 强调结合先验知识和数据，先验分布是推断的起点。 不使用先验信息，只考虑样本数据，避免主观先验的影响。
适用场景 适合小样本、动态更新、需要结合先验知识的场景，如医学诊断、机器学习、因果推断等。 适用于大样本、重复实验或没有先验知识的场景，如经典假设检验、实验设计等。
理论优势 结果直观，提供参数的完整概率分布，能够动态更新推断结果。 理论体系成熟，计算快捷，避免主观性，适合重复实验和大样本问题。
理论劣势 依赖先验分布，可能带来主观性；计算复杂，尤其在高维问题中。 无法量化参数的不确定性，难以处理小样本和动态问题。

12. 总体 (Population)

在统计学中，总体指的是研究中感兴趣的一组对象，通常是一群人（特别是在生物统计中）。

类似于集合，总体中包含的个体通常具有某些共同的特性，这些特性是研究的重点。例如： - 我的产品的用户群 - 糖尿病患者

研究论文通常会详细定义总体的特征，从而明确总体的范围： - 谁被纳入研究（包含标准）。 - 谁被排除在外（排除标准）。 不同总体的研究结果可能是不可推广的。例如，一个针对特定癌症患者的研究结果可能不能适用于所有癌症患者。

总体可能非常庞大（如数百万甚至数十亿人），由于时间和资源的限制，通常难以或不可能观察到总体中的每一个个体。由于无法直接研究整个总体，因此需要从总体中抽样 (Sampling)进行研究，比如 样本 (Sample) (x_1,x_2,\dots,x_n)。同时还会假设这些样本是由某种确定分布的随机变量 X 生成的，这就是统计模型 (Statistical model)。统计学的核心任务就是根据样本数据推断总体特征。

13. 参数 (Parameter) 与 估计量 (Estimator)

假设全体是所有30-40岁的男性的身高，统计模型为 X \sim N(\mu, \sigma^2)。参数 \mu, \sigma^2 可以看作是调节旋钮。原本要描述正态分布要对每个实数定义概率，而统计模型采用函数形式来描述它。只要参数确定，整个函数就确定了。而且统计参数往往带有特殊含义，描述了总体的某种特征。比如本例的参数就是全体身高的平均值和方差。

估计量是用来从样本数据推断 (Inference)总体参数的工具。大多数情况下是没有足够的资源来法观测全体中的每个实例的，所以模型参数是未知的。统计学的目标是用采样数据对未知的总体参数进行估计。比如根据目前收集到的身高样本 (x_1,x_2,\dots,x_n) 计算出样本均值 (Sample mean) \hat \mu=\frac 1 n \sum_{i=1}^n x_i 和样本方差 \hat \sigma^2，分别就是参数 \hat \mu, \hat \sigma^2 对应的估计量，也就是对参数的“猜测”或近似。一个好的估计量需要满足某些特性，其中最重要的是一致性（Consistency），即当样本量增加时，估计量的值会越来越接近真实的总体参数。如果我们能采集无限多的数据，估计量的值应该与真实参数完全重合。若估计量不一致，则不能使用，因为它无法可靠地反映总体参数。样本均值是一个优秀的估计量，因为它是一致的，并且在许多情况下表现良好。

属性 统计参数 估计量
对象 总体（未知） 样本（已知）
定义 描述总体特征的固定数值 基于样本数据的数学公式或规则
是否随机 固定值（未知，但确定存在） 随机变量（因样本不同而不同）
作用 是估计的目标 用于推测统计参数

推断统计学 (Inferential statistics)研究如何有效的估计参数。它有别于描述统计学 (Descriptive statistics)，后者只描述已收集的数据。

14. 大数定律 (Law of large numbers)

描述当收集大量样本后，样本均值 \hat \mu=\frac 1 n \sum_{i=1}^n x_i 跟全体均值产生大偏差的概率会越来越小：

\lim_{n\to \infty} P(|\hat \mu - \mu| \ge \epsilon) = 0

也记作 \hat \mu 依概率收敛 (Converges in probability)于 \mu。

\hat \mu \xrightarrow{p} \mu

这是一种新的极限概念。

15. 中心极限定理 (Central limit theorem)

由于采样操作具有随机性，所以基于这些样本的估计量是随机变量，其分布非常重要，称为样本分布 (Sampling distribution)。如果知道样本分布，就可以确定当前估计量的数值是正常的还是罕见的。

中心极限定理告诉我们，当样本量充分大时，样本均值的一个函数变换依分布收敛 (Converges in distribution)于标准正态分布：

\frac{\sqrt n} \sigma (\hat \mu - \mu) \xrightarrow{d} N(0,1)\;.

16. 渐进理论 (Asymptotic theory)

大数定律和中心极限定理都是渐进理论的例子。这也要求统计时尽可能获得充足的样本。当实在无法获得大量样本时，可以考虑一些其他方法。比如 Bootstrap 采样。

17. 假设检验 (Hypothesis tests)

统计学家将对总体的信念转化为关于总体参数的两个假设：

  • 第一个假设是 零假设 (Null hypothesis)，记作 H_0，表达我们希望推翻的假设，往往跟维持现状有关，比如：
    • 药物没有疗效
    • 实验组和对照组没有区别 这也体现了假设检验的严谨性：除非收集到足够证据，否则不能拒绝零假设。
  • 第二个假设是 备择假设 (Alternative hypothesis)，记作 H_a，是跟零假设相反的假设。

比如参数 \theta 表示实验组和对照组的区别，也是统计模型的参数。 H_0: \theta=0 零假设是二者没有区别。然后不断收集样本来计算估计量 \hat \theta。p 值 (p value) 就是在 H_0 成立的情况下，采样得到当前估计值以及更极端值的概率。如果这个概率足够低则表示 H_0 不太可能生成当前样本，可以拒绝 H_0。我们也可以通过置信区间 (Confidence interval) 来做决策。比如设置区间 (\hat \theta - c, \hat \theta + c)，如果 \theta=0 不在区间内，我们也可以拒绝 H_0。两种方法是等价的。

当我们做出决策后，可能会发生两类错误：

  • 第一类错误 (Type-I Error): H_0 其实是正确的，但我们却错误的拒绝了它。比如我们认为治疗是有效的，但其实是无效的。
  • 第二类错误 (Type-II Error): H_0 其实是错误的，但我们没能拒绝掉 H_0。比如我们把实际有效的药物当作无效的了。

理想中，我们应该同时降低这两类错误的概率，但实际中往往降低其中一个就会升高另一个。所以人们会先锁定可以容忍的第一类错误的概率上限，比如不高于 1% 。这称作显著度水平 (Significance level) 。之后再开始尝试最小化第二类错误。这个过程也称作最大化检验功效 (Maximizing power)。

18. z-检验 (z-test) 和 t-检验 (t-test)

根据研究问题的不同，假设检验的应用场景也有多种，比如：

  • 单样本检验 (One-sample test)：用于检验总体的特征，\hat \theta = \bar X 。
  • 两样本检验 (Two-sample test)：用于对比两组的差异，\hat \theta = \bar X_1 - \bar X_2 。

z-检验 和 t-检验都可用于单样本检验和两样本检验。如果要比较三组的差异，可以使用方差分析 (Analysis of variance) ，简称 ANOVA。比如 H_0 是三组的均值都一样，H_a 是其中一组的均值是不同的。

由中心极限定理，大样本下，样本分布将趋于正态分布：\bar X \sim N(\mu, \frac {\sigma^2} n), \quad \bar X_1 - \bar X_2 \sim N(\mu_1 - \mu_2, \frac {\sigma_1^2} n - \frac {\sigma_2^2} m) 。

当我们已知总体的标准差 \sigma 时，可以利用这一点进行 z-检验（这里的 z 有标准正态分布的含义），检验统计量 z = \frac {\hat \mu - \mu_0} {\sigma / \sqrt n} 其中 n 是样本量，\hat \mu 是样本均值，\mu_0 是总体均值的假设值。

如果样本量比较少 (n < 30)，或者不知道总体的标准差，则需要使用 t-检验，使用样本的标准差 s 作为总体标准差的估计。检验统计量 t = \frac {\hat \mu - \mu_0} {s / \sqrt n} t 服从自由度为 n-1 的 t 分布。当样本量较小时，t 分布比标准正态分布具有更重的尾部（即更高的极端值概率）。随着样本量的增加，t 分布逐渐逼近标准正态分布。

z-检验 和 t-检验都是针对连续型数据的。对于二元数据，且样本量较小时，可以使用列联表 (Contingency table)，并进行卡方检验 (\chi^2 test) 。

19. 回归 (Regression)

z-检验 和 t-检验之类的假设检验都是单变量分析 (Univariate analysis)，因为他们都聚焦于单个随机变量。如果要分析随机变量之间的关联，比如 Y=f(X)，就需要回归。

线性回归 (Linear regression)： Y=\beta_1 + \beta_2 X + \epsilon 让我们可以分析一个随机变量是如何影响另一个的。

广义线性回归 (Generalized linear regression): g(Y)=\beta_0+\beta_1 X 。线性回归只能处理连续型的输入，对于二元或计数类的数据，就需要用广义线性回归。

为了对回归方程中的参数进行估计：\hat \beta_0, \hat \beta_1, \hat \epsilon(\hat \sigma^2) ，我们需要用到 最大似然估计 (Maximum likelihood estimation) \hat \theta_\text{MLE} 或者其他最优化算法 (Optimization algorithm) 。

如果要了解随机变量随时间的变化，就需要对同一组对象在不同时刻采集数据。此时收集到的样本就不是 独立同分布 (Independent, Identically distributed) 的。这时应使用 纵向建模 (Longitudinal modeling)，比如 GEE model:

U(\beta) = \sum_{i=1}^n \frac {\partial \mu_i} {\partial \beta} V_i^{-1}(Y_i - \mu_i(\beta))

或 混合效应模型 (Mixed-effects model):

g(Y_i) = \beta_0 + \beta_1 X + (b_0 + b_1 X)

有时我们还希望在模型中包含多个预测因子：g(Y) = \beta_0+\beta_1 X_1 + \beta_2 X_2，那么就牵扯到变量选择 (Variable selection)，这可以咨询专家或者研究过去的数据来做选择。另一个经验法则是分析 干扰因素 (Confounders)，表示如果不计入某个变量，那么基于输入的预测将会被破坏。

20. 随机对照实验 (Randomized controlled trial)

数据的采集方式非常重要。使用随机对照实验获得的数据来分析，其结果可以被认为是有因果关系的，而不只是相关性关系，所以统计学家非常重视实验设计 (Experimental design)，这样我们才能准确的知道可以得出什么结论。

21. 非参数统计 (Nonparametric statistics)

在统计学中，必须始终牢记我们的假设，模型本身就是一种假设，假设了随机变量关联的形式。前面提到的模型中都包含参数，统称参数统计 (parametric statistics)。如果不想提前假设函数形式，而是直接估计整个函数 Y=f(X)，就可以使用非参数统计。比如 Mann-Whitney test 就是双样本 t-检验的非参数形式。

还有半参数统计 (Semiparametric statistics)，比较有名气的是 Cox model： \lambda(t | X)= \lambda_0(t) \exp(\beta_1 X_1)，用于生存分析 (Survival analysis)。此模型的 \exp 部分是参数化的，描述生存率如何随治疗而变化，而 \lambda_0 则是非参数的整个函数，成为风险函数 (Hazard function)。

22. 预测 (Prediction)

统计学的目标不只有描述和推断，还希望可以预测未来。相关的工具有机器学习 (Machine learning)，使用黑箱模型 (Black box models) 进行预测。在建模时，传统的假设是参数量远小于数据量，但在一些场合，会遇到参数量比数据量还大的情况，这就需要高维统计学 (High-dimensional statistics)。

23. 因果推断 (Causal inference)

一种让人们从观察的数据中做出因果陈述的技术。关键思想是反事实框架 (Counterfactual framework)。

24. 健壮统计 (Robust statistics)

统计往往依赖与假设，如果假设不成立，那么统计结论也就无效了。一些研究者开发了健壮统计技术，使得即使假设是错误的，也能做出正确的推理，称作假设松弛 (Assumption relaxation)。
